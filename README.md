# music-bot
🎧 Music Search Engine Bot  A fast and minimal Telegram music discovery bot.  Search music across multiple platforms in one place.

---

## 🚀 Features

- 🎵 YouTube Music search
- ☁️ SoundCloud discovery
- 🎸 Bandcamp search
- 🎼 MusicBrainz database (rare tracks)
- ⭐ Telegram Stars donations support
- ⚡ Fast and clean UI

---

## 📦 Commands

- /start — Start bot
- /search — Find music
- /rare — Underground tracks
- /mood — Music by mood
- /donate — Support project ⭐
- /about — Info
